# Berries
Berries allow you to remove or randomize different traits of your monster, and can be used in the Apothecary.

| Item Name        | Effect                                                                          |
|------------------|---------------------------------------------------------------------------------|

| Bugger Berry     | Remove Species 1                                                                |

| Cocon Berry      | Randomize Type 2 (if present)                                                   |
| Datei Berry      | Randomize Attribute                                                             |

| Lilan Berry      | Remove Type 2 (if present)                                                      |
| Maizi Berry      | Remove Type 4 (if present)                                                      |
