# Egg Items
Egg Items allow you to modify the outcome of your egg roll, from garuntees to exclusions, assuming the code behaves, there's plenty of control for all your specifc wants.

| Item Name                       | Effect                                                                                              |
|---------------------------------|-----------------------------------------------------------------------------------------------------|
| Incubator                       | Allows you to skip the creative challenge.                                                          |
| [type] Nurture Kit              | Sets Primary Type Guarantee to that Type.                                                           |
| Corruption Code                 | Sets Attribute to Virus.                                                                            |
| Repair Code                     | Sets Attribute to Vaccine.                                                                          |
| Shiny New Code                  | Sets Attribute to Data.                                                                             |
| [Rank] Rank Insense             | Filters to limit to that rank of Yokai.                                                             |
| [Yokai Attribute] Color Insense | Filters to limit to that Attribute of Yokai.                                                        |
| Spell Tag                       | Excludes Yokai.                                                                                     |
| DigiMeat                        | Includes Rookie Digimon.                                                                            |
| DigiTofu                        | Excludes Digimon.                                                                                   |
| Broken Bell                     | Excludes Pokemon.                                                                                   |
| [type] Poffin                   | Filters to only Pok√©mon of these type(s).                                                           |
| # [attribute] tag               | Filters to only Digimon of these attribute(s).                                                      |
| DNA Splicer                     | Allows the claiming of an extra mon per DNA Splicer used.                                           |
| Hot Chocolate                   | Guarantees Fusions.                                                                                 |
| Vanilla Milk                    | Guarantees 2+ types.                                                                                |
| Chocolate Milk                  | Guarantees 3+ types.                                                                                |
| Strawberry Milk                 | Guarantees 4+ types.                                                                                |
| MooMoo Milk                     | Guarantees 5 types.                                                                                 |
| Vanilla Ice Cream               | Guarantees 1st Type.                                                                                |
| Strawberry Ice Cream            | Guarantees 2nd Type (can only be used if using Vanilla Ice Cream).                                  |
| Chocolate Ice Cream             | Guarantees 3rd Type (can only be used if using Vanilla and Strawberry Ice Cream).                   |
| Mint Ice Cream                  | Guarantees 4th Type (can only be used if using Vanilla, Strawberry, and Chocolate Ice Cream).       |
| Pecan Ice Cream                 | Guarantees 5th Type (can only be used if using Vanilla, Strawberry, Chocolate, and Mint Ice Cream). |
| Input Field                     | Sets Species 1 to inputted data (Species 2 and 3 still roll normally).                              |
| Drop Down                       | Sets Species 1 and 2 to only roll from two species (inputted).                                      |
| Radio Buttons                   | Sets Species 1, 2, and 3 to only roll from three species (inputted).                                |
